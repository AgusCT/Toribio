Module openmoko

Library for managing a OpenMoko smartphone.

At the moment, supports FreeRunner under SHR-testing.

Functions

usb_mode (mode)
Set USB mode.

Parameters:

  • mode: Optional, either 'host' or 'device' changes the mode.

Returns:

    The mode as set.
usb_power_direction (direction)
Set USB power.

Parameters:

  • direction: optional. If 'out', the phone is powering the connected devices. If 'in' the phone charges from the port.

Returns:

    The direction as set.
usb_currlim (currlim)
Set the current limit on the USB port. When powered from an "dumb" device, the phone sets a 100mA limit by default.

Parameters:

  • currlim: Optional the current limit in mA. Supported values are 0, 100, 500 and 1000.

Returns:

    The current limit as set.
